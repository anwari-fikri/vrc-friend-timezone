# vrc-friend-timezone

Keep track of your friends' timezones.

### What it does

- Add your friends to a list with name, image, timezone, and birthday.
- Everything is stored locally in localStorage, so no login required.
- See their local time instantly, (daylight savings adjustment included)

### Why I made it

- So I can greet my friends on their birthday **right at 12 AM**
- Or “judge their sleeping time” (jk)… really just to see if they’re awake or not.

### How to use

- I have hosted this app on vercel
- Try it here: [vrc-friend-timezone](vrc-friend-timezone.vercel.app)
